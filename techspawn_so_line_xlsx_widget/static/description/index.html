Product Template in Sale Order Line

Settings >> General Settings >> Sale

Here we can enable "Product Template in SO Line."

If "Product Template in SO Line" is enabled, we can see the product will come from Product Template.

If "Product Template in SO Line" is disabled, we can see the product will come from Product Product(Variants).

Send Sale Order Excel Report By Email

If we clicked the "Send Excel Report By Email" button, a wizard will be popup.

In this wizard we can see the attached Excel report. By clicking the send button, the email will be sent with the attached excel report. We can also download the Excel report from this wizard.

This is the Sale Order Excel Report.
